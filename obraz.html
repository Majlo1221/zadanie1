<  input type="button" name="przycisk" value="Nowa Strona" onclick="WinOpen_bembenista('')"  >
type = button -> przycisk
name = przycisk -> nazwa dla nas
value = Nowa strona -> to co się wyświetla na przycisku w stronie
onclick = WinOpen_bembenista('') -> co ma robić przycisk, zdefiniowane w linijkach kodu od 5 do 8 w przyklad1a
window.open("obraz.html","okienko","toolbar=no,directories=no,menubar=no,height=280,width=160,top=200,left=200");
jest to okno które wyskakuje po kliknięciu na przycisk.
1. "obraz.html" -> definiujemy do czego ma to być odnośnik
2. "okienko" -> nazwa dla nas
3. "toolbar=yes/no -> pasek narzędzi
4. directories=yes/no -> katalogi
6. menubar=yes/no -> pasek menu
7. height=280 -> wysokośc okienka
8. width=160 -> szerokośc okienka
9. top=200 -> margines od góry
10. left = 200 -> margines od lewej
11. window.cose() -> zamykanie okna, definiuje się w funkcji od 4 do 7 inijki w tym kodzie
